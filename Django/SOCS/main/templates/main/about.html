About SOCS

The Space Operation & Control System (SOCS) is just as it sounds,tools to operate and control Milton Keynes Makerspace. It is currently a work in progress and has very few features. Future features will hopefully allow for membership management, access to various tools such as issue reporting as well as live data streams from the space. It will also provide a connection to external systems such as SpaceAPI.

SOCS is currently under development, but do feel free to get in contact if you would like to get involved

The Source code will be open source and available on Github, made up of various parts. This site acts as a front end, while there will be various backend systems such as MQTT etc.

Current Ideas include;

  • Membership Management - Donations & Payments
  • Issue Reporting - Tools fixing
  • Membership & Asset logging
  • Project Storage Label Production
  • Provide SpaceAPI endpoint
  • Data & Control - MQTT to control things in the space, collect data from such as temp etc.
  • Integrate with AccessControl - Octoprint (Manage access rights for all printers from one place)
